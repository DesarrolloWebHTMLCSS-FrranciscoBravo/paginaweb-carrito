activadores

Avalanche requiere una licencia para cada dispositivo móvil que administra. Cuando un servidor detecta un dispositivo nuevo, se envía una solicitud de licencia al Servidor de licencias. Este servidor, envía una licencia al servidor para que la distribuya. El archivo de licencia es único para el servidor y no se puede transferir a otro servidor. Cuando el servidor recibe la licencia, Avalanche puede administrar ese dispositivo. Si una licencia caduca o se libera, ésta vuelve al grupo de licencias del Servidor de licencias, hasta que otro servidor la solicite.
